About Secure File Locker

Secure File Locker was created by Derrick Richard to help users encrypt files into self-unlocking HTML pages. Simply upload a file, set a password, and optionally choose an expiration date. The tool generates a standalone HTML file that can decrypt itself in any browser — no software or server required.
