FileLock

Encrypt files into self-unlocking HTML pages

About FileLock

FileLock was created by Derrick Richard to help users encrypt files into self-unlocking HTML pages. Simply upload a file, set a password, and optionally choose an expiration date. The tool generates a standalone HTML file that can decrypt itself in any browser. No software or server required.
